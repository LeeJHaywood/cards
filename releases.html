Release history


v3.2   13 Jun 2018

  • Added missing check to decoder tab to ignore user selections of placeholder images.


v3.1   12 Jun 2018

  • Amended image pre-loading, which had been preventing selection of cards in the decoder tab.


v3.0   7 May 2018

  • Added bit-level encryption of message to correct flaw when both sides are being used, with automatic detection by decoder.
  • Added proper detection of zero bytes with smarter escaping.
  • Improved performance by detecting when input stops changing.
  • Improved detection of messages that are too long by adding an explicit length check.
  • Added link/QR code processing in downloadable version.  Links still relate to leehaywood.org rather than the new server.
  • Added this release history/download page.


v2.2   1 Apr 2018

  • Added alert box warning about known flaw when use of both sides is first selected.
  • Removed download link from the downloaded archive itself.


v2.1   27 Mar 2018

  • Added trivial escaping of messages ending with 'su'.
  • Added checkbox to reveal numeric encodings at bottom of page, with an alert box warning not to share the values.
  • Added list of known issues to technical page.


v2.0   23 Mar 2018

  • Added link/QR code tab.  Please note that this functionality was not included in the downloadable version until version 3.0.


v1.0   4 Oct 2014

  • Original version.
